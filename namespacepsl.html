Enum documentation

enum psl::enum_ops_t
#include <enum.hpp>

Defines the set of operations that can be applied to the given enum.

Enumerators
NONE

No operations permitted.
BIT

bit operators are permitted

operators &, |, ^, ~, &=, |=, and ^=
SHIFT

shift related operators are permitted

operators <<, >>, <<=, and >>=
ARITHMETIC

remaining arithmetic operators that do not fall in the previous categories

operators +, -, /, *, %, +=, -=, /=, *=, and %=, for both the enum's underlaying type as well as any type that can normally do these actions with the underlaying type
LOGICAL

defines the logical bool operator&& (AND) to return true when the left-hand side satisfies the flag of the right-hand side

Warning

do take special care when enabling this as it changes the behaviour of the logical and operator, which might be confusing
