    template<typename T, typename Error = std::error_code>
    psl::expected class

    contains either a result (the expected), or an error

    Template parameters
    T value type to contain.
    Error error type to throw, by default this is std::error_code.

    Contents

    • Public types
    • Constructors, destructors, conversion operators
    • Public functions

    Models an object that can either contain the result of an operation, or an error in case the operation failed. It cannot be re-assigned, meaning that the value or error is set at construction time. After this it supports continuation behaviours then() and on_error().
