    psl::dynamic_sbo_storage struct

    Uses sbo_storage when available, otherwise will default to using an allocator.

    Template parameters
    T element type to store
    SBO_count
    Allocator Allocator to use as fallback
    Alias should the SBO alias or not? read psl::details::sbo_storage for more info.

    Warning

    This type does not run destructors for its contained elements, it's up to whoever uses it to make sure the memory is properly cleared before deallocation happens.
